# ✨ Features

- 📁 Professional File Explorer
- 📂 Create & Manage Files/Folders
- 📝 Multi-language Code Editing
- 🔍 Instant File Search
- ▶️ One-click Code Execution
- 🌐 Live HTML/CSS/JavaScript Preview
- 🐍 Python Execution (Pyodide)
- 💾 Auto Save with Local Storage
- 📥 Import Existing Files
- 📤 Export Files
- 📦 Download Entire Project
- ✏️ Rename Files & Folders
- 🗑 Delete Files & Folders
- 📋 Copy File Paths
- ⌨️ Keyboard Shortcuts
- 📱 Responsive Design
- 🌙 Modern Dark UI

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/EliteProTech/FECT-Code-Editor.git
```

Enter the project directory

```bash
cd FECT-Code-Editor
```

Open **index.html** in your browser or run a local web server.
